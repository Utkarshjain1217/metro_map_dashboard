🚇 Metro Map Dashboard

An interactive web-based dashboard to visualize metro routes, Origin-Destination (OD) pairs, and traffic insights. This project helps analyze mobility patterns using maps, filters, and dynamic charts.

📌 Features
🗺️ Interactive metro map visualization
🔍 OD (Origin-Destination) pair analysis
📊 Dynamic charts for traffic insights
🎛️ Filters for customized data exploration
⚡ Lightweight and fast frontend
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Visualization: Custom JS + Chart-based rendering
Mapping: JavaScript-based map integration
